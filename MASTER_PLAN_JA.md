## 3. リスク分析

### 収入リスク

| リスク | 確率 | 影響度 | 対策 |
|--------|------|--------|------|
| DataAnnotation/Outlier不採用 | 中 | 高 | Appen、Telus、Scale AI、Surge AIにも同時応募 |
| フリーランス立ち上がりが遅い | 中 | 中 | レイヤー1を最低ラインとして維持、実績構築のため最初は低単価で受注 |
| SaaSがPMFを見つけられない | 中 | 低（他レイヤーでカバー） | 構築前にTwitter/Redditでアイデア検証、素早くピボット |
| クライアント支払い遅延 | 低 | 中 | エスクロー付きプラットフォーム（Upwork）を利用、直接案件は50%前金 |
| AIがアノテーション業務を代替 | 中（12〜18ヶ月） | 高 | レイヤー3はこのためにある — 代替される前に移行 |

### ビザリスク

| リスク | 確率 | 影響度 | 対策 |
|--------|------|--------|------|
| DE Rantau申請却下 | 低 | 高 | 日本パスポートで90日ビザなし入国可、マレーシア国内から再申請 |
| 収入証明不十分 | 低 | 中 | 早期から稼ぎ始め、3ヶ月以上の明細を維持 |
| プログラム廃止/変更 | 極低 | 高 | 代替: MM2H、自社設立でEmployment Pass、ビザラン |
| パートナーのビザ問題（未婚の場合） | 中 | 高 | 申請前に入籍 — 全てが簡単になる |

### プラットフォーム依存リスク

| リスク | 対策 |
|--------|------|
| Upworkアカウント制限 | 分散: Toptal、直接クライアント、LinkedIn営業 |
| DataAnnotation報酬率低下 | レイヤー2&3への移行中 |
| Gumroad手数料値上げ | Stripe + LemonSqueezyでセルフホストにバックアップ |
| 単一SaaS製品の失敗 | デジタル商品が並行収益を提供 |

### バックアッププラン

1. **9月時点で貯蓄が¥800,000未満の場合**: 移住を2027年1月に3ヶ月延期
2. **ビザ却下の場合**: ビザなしでマレーシア入国（90日間）、KLから再申請
3. **MY到着後の収入が$2,500/月以下の場合**: 緊急フリーランス発動（案件選ばず）、支出を$1,500/月に削減（KLでは十分可能）
4. **最終手段**: 日本に帰国、就職復帰、6ヶ月後に再挑戦

## 4. ウェブアプリ設計

### アプリ名: **作戦（Sakusen）**

### 技術スタック

| レイヤー | 選択 | 理由 |
|---------|------|------|
| フレームワーク | Next.js 14+（App Router） | フルスタック、高速、優れたDX |
| 言語 | TypeScript | 型安全、得意分野 |
| データベース | PostgreSQL（Supabase経由） | 無料枠、認証内蔵、リアルタイム |
| 認証 | Supabase Auth | マルチユーザー、メール/パスワード、無料 |
| ORM | Prisma | 型安全なDBクエリ |
| スタイリング | Tailwind CSS + shadcn/ui | 高速、美しい、コピペコンポーネント |
| ホスティング | Vercel | 無料枠、Gitから自動デプロイ |
| ファイル保存 | Supabase Storage | ドキュメントアップロード用 |
| チャート | Recharts | 収入ダッシュボードの可視化 |

**合計コスト: $0/月**（全て無料枠）

### データベーススキーマ

```sql
-- ユーザー（あなたとパートナー）
CREATE TABLE users (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  email TEXT UNIQUE NOT NULL,
  name TEXT NOT NULL,
  role TEXT DEFAULT 'member', -- 'admin' | 'member'
  avatar_url TEXT,
  created_at TIMESTAMPTZ DEFAULT now()
);

-- 共有チェックリスト
CREATE TABLE tasks (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  title TEXT NOT NULL,
  description TEXT,
  category TEXT NOT NULL, -- 'visa' | 'income' | 'life' | 'tech'
  priority TEXT DEFAULT 'medium', -- 'critical' | 'high' | 'medium' | 'low'
  status TEXT DEFAULT 'todo', -- 'todo' | 'in_progress' | 'done' | 'blocked'
  due_date DATE,
  assigned_to UUID REFERENCES users(id),
  created_by UUID REFERENCES users(id),
  depends_on UUID REFERENCES tasks(id),
  created_at TIMESTAMPTZ DEFAULT now(),
  updated_at TIMESTAMPTZ DEFAULT now()
);

-- 収入追跡
CREATE TABLE income_entries (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  source TEXT NOT NULL, -- 'dataannotation' | 'freelance' | 'saas' | 'digital_products' | 'other'
  amount DECIMAL(10,2) NOT NULL,
  currency TEXT DEFAULT 'USD', -- 'USD' | 'JPY' | 'MYR'
  layer INTEGER NOT NULL, -- 1, 2, or 3
  description TEXT,
  date DATE NOT NULL,
  user_id UUID REFERENCES users(id),
  created_at TIMESTAMPTZ DEFAULT now()
);

-- 月間収入目標
CREATE TABLE income_goals (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  month DATE NOT NULL, -- 月の初日
  target_amount DECIMAL(10,2) NOT NULL,
  currency TEXT DEFAULT 'USD',
  created_at TIMESTAMPTZ DEFAULT now()
);

-- ビザ進捗
CREATE TABLE visa_steps (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  title TEXT NOT NULL,
  description TEXT,
  status TEXT DEFAULT 'pending', -- 'pending' | 'in_progress' | 'completed' | 'blocked'
  due_date DATE,
  completed_at TIMESTAMPTZ,
  notes TEXT,
  sort_order INTEGER NOT NULL,
  created_at TIMESTAMPTZ DEFAULT now()
);

-- ドキュメント
CREATE TABLE documents (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  name TEXT NOT NULL,
  category TEXT NOT NULL, -- 'visa' | 'financial' | 'personal' | 'business'
  file_url TEXT NOT NULL,
  file_type TEXT, -- 'pdf' | 'image' | 'doc'
  uploaded_by UUID REFERENCES users(id),
  notes TEXT,
  created_at TIMESTAMPTZ DEFAULT now()
);

-- 意思決定ログ
CREATE TABLE decisions (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  title TEXT NOT NULL,
  context TEXT, -- この決定が必要になった背景
  options TEXT, -- 検討した選択肢のJSON配列
  chosen TEXT NOT NULL, -- 選んだもの
  reasoning TEXT, -- この選択肢を選んだ理由
  decided_by UUID REFERENCES users(id),
  decided_at TIMESTAMPTZ DEFAULT now(),
  category TEXT -- 'visa' | 'income' | 'life' | 'tech'
);

-- リサーチノート
CREATE TABLE notes (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  title TEXT NOT NULL,
  content TEXT NOT NULL, -- markdown
  category TEXT, -- 'visa' | 'housing' | 'income' | 'lifestyle' | 'tech'
  tags TEXT[], -- タグの配列
  created_by UUID REFERENCES users(id),
  pinned BOOLEAN DEFAULT false,
  created_at TIMESTAMPTZ DEFAULT now(),
  updated_at TIMESTAMPTZ DEFAULT now()
);

-- 目標
CREATE TABLE goals (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  title TEXT NOT NULL,
  description TEXT,
  category TEXT NOT NULL, -- 'financial' | 'visa' | 'business' | 'personal'
  target_value DECIMAL(10,2), -- 例: 貯蓄目標
  current_value DECIMAL(10,2) DEFAULT 0,
  unit TEXT, -- 'USD' | 'JPY' | 'count' | 'percent'
  deadline DATE,
  status TEXT DEFAULT 'active', -- 'active' | 'completed' | 'paused'
  created_at TIMESTAMPTZ DEFAULT now()
);
```
